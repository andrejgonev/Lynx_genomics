### Command for filtering the barcodes
# I used the barcodes_vertical file from Enrico, grep all the eurasian lynx samples (c_ll), transform them by removing all unnecessary symbols
# I printed the output by taking the second column as the first, and then the first column twice + adding the name extensions for the R1 and R2 respectively.
# Finally, I removed (inverese grep) all the samples that Enrico has already processed with fastp
# The file is saved as barcodes_filtered
